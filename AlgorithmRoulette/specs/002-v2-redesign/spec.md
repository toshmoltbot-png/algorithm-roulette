# Algorithm Roulette V2 - UI Redesign

## Core Concept
We are doing a total visual teardown and rebuild of the "Algorithm Roulette" landing page (the wheel page). The current V1 is a functional but basic React component with a flat, 8-segment pie chart on a dark teal background. 

The V2 needs to feel like a high-end, slightly dystopian, cyberpunk portal. It should be mobile-first and look native-app quality, preparing the user to drop into an algorithmic echo chamber (TikTok/Shorts style).

## Scope
This spec only covers the visual redesign of the Wheel landing page. 
- The routing, backend, and actual video feed are out of scope.
- We are replacing the existing Wheel component and Landing page layout with a new, highly-polished Framer Motion + Tailwind implementation.

## Requirements
1. **Dark Mode & Neon Glows:** Use `slate-950` with subtle cyan/magenta absolute positioned blurs.
2. **Glassmorphic Wheel:** Replace the flat pie chart with a 3D-styled, gradient-rich wheel using `framer-motion` for the spinning physics.
3. **Tactile CTA:** The "SPIN" button needs to look incredibly satisfying to press (glows, scaling, active states).
4. **Responsive:** Must look perfect on a mobile viewport (iPhone 14/15 size) while centering nicely on desktop.
